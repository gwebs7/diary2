알림 메시지

클라우드 연동 가이드

STEP 1

HTML 소스코드 하단의 const firebaseConfig 구역에 Firebase 프로젝트 콘솔에서 발급한 앱 구성 데이터(API 키 등)를 정확하게 삽입했는지 확인해 주세요.

STEP 2 (중요)

Firestore Database 메뉴의 규칙(Rules)이 아래 구조와 다르게 기본값(false)으로 설정되어 있다면, 아래 내용으로 바꿔서 [게시] 버튼을 눌러주셔야 쓰기 권한이 허용됩니다.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
STEP 3

Authentication -> Sign-in method 탭 -> 익명(Anonymous) 로그인이 활성화되어 있어야 에러가 나지 않고 인증 키를 받아올 수 있습니다.
